Post Pal

Developed a full-stack blog application using Flask, SQLAlchemy, and Bootstrap, featuring user authentication, post creation, and commenting functionality. Implemented role-based access control (RBAC) to restrict post creation and deletion to verified users only. Integrated CKEditor for rich text editing and Gravatar for user avatars in comments. Designed and implemented a relational database schema with SQLAlchemy ORM, including models for Users, BlogPosts, and Comments, ensuring referential integrity with foreign key constraints. Utilized Flask-Login for managing user sessions and Flask-WTF for form handling Stored salted and hashed passwords with Python-Werkzeug to ensure user safety.
